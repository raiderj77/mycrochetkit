# Implementation Plan: Enhancing My Crochet Kit

This plan outlines the steps to elevate My Crochet Kit (MCK) to a market-leading position by addressing feature gaps, improving UX, and ensuring transparency.

## 🟢 Phase 1: Immediate Enhancements & Transparency (Current)
- [x] **Clarify Referral Terms**: Multi-referral requirement and US-only policy added to all program pages.
- [x] **Risk-Free Guarantee**: 30-day money-back guarantee prominently displayed.
- [x] **PWA Optimization**: Install prompt throttled to once per session.
- [x] **Public Calculator**: Yarn calculator moved to marketing layout for gated-free use with Signup CTA.
- [x] **Dynamic Founders Counter**: Ensure real-time Firestore sync for the 500 spots countdown.
- [x] **Initial Annotation Tools**: PDF Sticky notes and row highlighter implemented.

## 🟡 Phase 2: Advanced Pattern & Stash Tools
- [x] **Comprehensive Stash Schema**: Added fields for fiber type, dye lot, purchase details, and care instructions to `AddYarnDialog`.
- [x] **Integrated Costing**: Automatically pull yarn costs from stash into the project pricing calculator.
- [x] **Refined Annotations**: Multi-color highlight support and digital sticky notes implemented.
- [x] **Advanced Pattern Tools**: Added "Repeat Counters" and "Quick-Ref Overlay" to the PDF viewer.
- [x] **Project Kanban**: Implemented a "View Mode" toggle in Project List with a dedicated Kanban stage dashboard.

## 🔵 Phase 3: Accessibility & UX Polish
- [x] **Accessibility Audit**: Initial sweep completed (Hero ARIA labels, decorative elements hidden). Ongoing WCAG AA compliance.
- [x] **Built-in Tutorials**: Patterns in the library now serve as built-in tutorials for new users.

## 🟣 Phase 4: Growth & Innovation
- [x] **Starter Pattern Library**: Curate and embed free beginner patterns to reduce onboarding friction.
- [ ] **Gamification**: Implement achievement badges for project milestones.
- [ ] **Smart Integrations**: Research Apple Watch and Siri/Google Assistant shortcuts.
